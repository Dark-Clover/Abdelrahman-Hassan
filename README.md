# Abdelrahman Hassan - IBM Django Final Project

Final project submission for **Django Application Development with SQL and Databases** (IBM Full Stack Software Developer Certificate).

## Submission Files

| Task | File | Description |
|------|------|-------------|
| Task 1 | `onlinecourse/models.py` | Question, Choice, and Submission models |
| Task 2 | `onlinecourse/admin.py` | Seven imported classes, QuestionInline, ChoiceInline, QuestionAdmin, LessonAdmin |
| Task 4 | `onlinecourse/templates/onlinecourse/course_details_bootstrap.html` | Course detail template with lessons and exam form |
| Task 5 | `onlinecourse/views.py` | Includes `submit` and `show_exam_result` functions |
| Task 6 | `onlinecourse/urls.py` | Includes routes for `submit` and `show_exam_result` |
